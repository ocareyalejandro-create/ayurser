# Ayurser

*Life-being.* A calm, one-minute Ayurvedic daily check-in.

Answer five gentle questions, receive one considered guidance card — a focus, a food, a ritual, a breath, a movement — grounded in dosha (Vata / Pitta / Kapha).

> Lifestyle guidance inspired by Ayurveda — not medical advice.

Built by Alex & Vanessa. v0.1 prototype.
